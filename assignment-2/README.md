\##  Steps to Deploy on Polygon Testnet



1\. Open Remix IDE: https://remix.ethereum.org

2\. Create a new file `contract.sol`

3\. Write the smart contract code

4\. Compile using Solidity compiler (version 0.8.x)

5\. Open \*\*Deploy \& Run Transactions\*\* tab

6\. Select \*\*Injected Provider - MetaMask\*\*

7\. Connect MetaMask wallet

8\. Select \*\*Polygon Amoy Testnet\*\* in MetaMask

9\. Click \*\*Deploy\*\*

10\. Confirm the transaction in MetaMask

11\. After deployment, copy the contract address

12\. Verify the contract on PolygonScan
