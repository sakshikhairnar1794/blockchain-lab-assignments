\# Assignment 1 - Smart Contract Development



\## Purpose

This smart contract is used to store and retrieve a number on the blockchain.



\## Functions

\- set(uint \_data): Stores a value

\- get(): Returns the stored value



\## Tools Used

\- Solidity

\- Remix IDE



\## Steps to Run

1\. Open Remix IDE

2\. Create contract.sol file

3\. Paste the smart contract code

4\. Compile using Solidity compiler

5\. Deploy using Remix VM

6\. Test set and get functions
